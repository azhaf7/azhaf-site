# Portfolio Website

A modern, responsive portfolio website showcasing my projects, skills, and experience as a software developer.

## Features

- **Dark/Light Theme Toggle** - Switch between dark and light modes with system preference detection
- **Responsive Design** - Fully responsive layout that works on all devices
- **Interactive Sections**:
  - Hero section with animated code examples
  - About section with animated statistics
  - Skills section with progress bars
  - Projects section with expandable project cards
  - Hobbies section with Super Mario World-inspired design
  - Contact form with code-style interface
- **Smooth Animations** - Scroll-triggered animations and transitions
- **Modern UI** - Clean, minimal design with a focus on content

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Press Start 2P for Mario section)

## Getting Started

### Prerequisites

No build tools or dependencies required! This is a static website that can be opened directly in a browser.

### Installation

1. Clone the repository:
```bash
git clone https://github.com/azhaf7/portfolio-website.git
cd portfolio-website
```

2. Open `index.html` in your web browser, or use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server
```

Then navigate to `http://localhost:8000` in your browser.

## Customization

### Updating Personal Information

Edit `index.html` to update:
- Name and title
- Bio and description
- Skills and technologies
- Projects
- Contact information
- Social media links

### Changing Colors and Themes

Edit CSS variables in `styles.css`:
```css
:root {
    --color-bg: #ffffff;
    --color-text: #0f172a;
    /* ... more variables */
}
```

### Adding New Projects

Add new project cards in the Projects section of `index.html` following the existing structure.
